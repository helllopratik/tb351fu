Current Projects

Two repos are ready to use right now

These are the two public repos people will need most often for TB351FU work. One covers the unlock route, and the other helps turn Lenovo firmware packages into something easier to work with during flashing.

TB351FU Bootloader Linux

Bootloader Unlock

Bootloader unlock steps, token handling and verification notes for people working from Linux on the Lenovo Tab Plus TB351FU.

  • Device-specific unlock token generation.
  • Guidance around the required auth and flash assets.
  • Fastboot checks after the unlock step is complete.
Open GitHub Repo
Firmware SP Flash Tool Metadata

Lenovo Flash XML Helper

Utility for decrypting Lenovo firmware metadata and preparing a cleaner SP Flash Tool package for TB351FU restores, testing and device-side work.

  • Decrypts Lenovo .x firmware metadata.
  • Checks the XML structure and referenced image files.
  • Builds a flash-ready sp_flash_tool_bundle/.
