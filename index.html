Workflow

How the tools fit together

Visitors should quickly understand that your repositories are not random uploads. They form a connected workflow around Lenovo firmware preparation and bootloader unlocking.

01

Collect the official package

Start with the correct TB351FU firmware package and confirm the device-specific files match the target tablet.

02

Decrypt Lenovo metadata

Use the XML helper to convert encrypted .x files into usable XML, then validate the package structure.

03

Generate unlock data

Move into the bootloader unlock repo to run the guided token generation stage with mtkclient.

04

Flash and verify

Load the prepared flash assets, complete the final unlock routine, and confirm the bootloader state in fastboot.
